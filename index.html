Projects

Homelab

My pride and joy, my homelab is always a work in progress to self-host everything from my website to backups.

Website

This website is self-hosted, built by HTML5 and CSS, on top of a LEMP (Linux, Nginx, MariaDB, PHP) stack, and includes a documentation wiki with a blog.

Media Server

No longer wanting to rely on streaming services, I crafted a media server using Plex and a Sonarr, Radarr, SABnzb, Deluge stack to download and organize the media. It has allowed friends and family to stream high quality content to any device in or outside my home network.

Remote Server - In Progress

Using an old ThinkPad laptop, a Raspberry Pi, and a few network switches, it opens a secure WireGuard VPN tunnel to rsync scheduled cron scripts into a LVM LUKS encrypted backup drive for remote backups. It also serves as a Plex media server. It is still a work in progress.

Timeshift Script

A bash script written for the pacman package manager with a hook to automatically snapshot the system using Timeshift before any updates. Includes a configuration file.

Automated Backups

Following the 3-2-1 Backup Strategy, I utilized programs such as rsync, cronjobs, ghettoVCB, and bash/shell scripting to design a fully automated backup system for my servers. It runs on a weekly and monthly schedule, fully logging any processes.

Network Map

Using draw.io, I created a networking diagram for my homelab, parent's home network, and the home I currently live at. It has allowed me to visualize my network in a way that allows me to see a bigger picture of what's going on and to plan for future projects.

BookStack

A free and open source self-hosted wiki platform built on a LEMP (Linux, Nginx, MariaDB, PHP) stack allows me to organize and store my personal documentation in a nice and simple solution. Bonus : Cookbook!
