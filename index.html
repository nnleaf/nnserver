Projects

Homelab

My pride and joy, my homelab is always a work in progress to self-host everything from my website to backups.

Network Map

Using draw.io, I created a networking diagram for my homelab, home network, and the room I rent. It has allowed me to visualize my network to see what's going on and to plan for future projects.

unRAID Server

I needed a dedicated NAS that provided data parity to mitigate disk failure issues, so I went with unRAID to buiild a new server. Migrating all my services was the largest challenge, but by planning out the migration, it alleviated a lot of the issues I ran into and lowered downtime.

Website

This website is self-hosted, built by HTML5 and CSS, on top of a LEMP (Linux, Nginx, MariaDB, PHP) stack, and includes a documentation wiki with a blog.

Remote Server

Using an old ThinkPad laptop, a Raspberry Pi, and a few network switches, it opens a secure WireGuard VPN tunnel to rsync scheduled cron scripts into a LVM LUKS encrypted backup drive for remote backups. It also serves as a Plex media server and an offline backup of BookStack with my documentation.

Timeshift Script

A bash script written for the pacman package manager with a hook to automatically snapshot the system using Timeshift before any updates. Includes a configuration file.

Automated Backups

Following the 3-2-1 Backup Strategy, I utilized programs such as rsync, cronjobs, ghettoVCB, Veeam and bash/shell scripting to design a fully automated backup system for my servers. It runs on a weekly and monthly schedule, fully logging any processes.

Grafana Monitoring

Utilizing Grafana, Telegraf, and InfluxDB, I aimed to centralize all monitoring and graphing to one dashboard in order to easily monitor and analyze my server and services.

BookStack Documentation

A free and open source self-hosted wiki platform built on a LEMP (Linux, Nginx, MariaDB, PHP) stack allows me to organize and store my personal documentation in a nice and simple solution. Bonus : Cookbook!

Media Server

No longer wanting to rely on streaming services, I deployed a media server using Plex and a Sonarr, Radarr, SABnzb, Deluge stack to download and organize the media. It has allowed friends and family to stream high quality content to any device in or outside my home network.
